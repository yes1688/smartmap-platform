## 🏗️ 專案技術架構

### 後端技術棧
- **Go 1.21+** - 主要程式語言
- **Gin** - Web 框架
- **PostgreSQL + PostGIS** - 空間資料庫
- **Ollama** - 本地 LLM 服務
- **WebSocket** - 即時通訊
- **GORM** - ORM 資料庫操作

### 前端技術棧
- **CesiumJS** - 3D 地圖渲染引擎
- **WebRTC** - 語音識別 API
- **WebSocket** - 即時通訊
- **Vanilla JavaScript** - 原生前端開發
- **SolidJS** - 新前端框架 (實驗性)

### 基礎設施
- **Podman + Podman Compose** - 容器化部署
- **Nginx** - 反向代理與靜態檔案服務
- **多階段構建** - 優化容器映像大小
